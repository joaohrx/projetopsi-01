# Livraria Marginália
Esse repositório é destinado para o 1º projeto de PSI do ano, que consiste em criar uma aplicação web funcional básica.
Nosso projeto busca reutilizar o projeto anterior em Design Web de uma livraria, construindo um serviço de compra, empréstimo e exibição de livros, visando facilitar e aumentar o acesso popular à literatura.
